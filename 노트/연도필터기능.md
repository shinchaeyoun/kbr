검색 창에 셀렉트 박스 추가.   
연도 선택 셀렉트 박스.   
예를 들어. '2025 / 건강'으로 검색할 경우    
2025년도의 건강 정보만 불러올 수 있도록.


### 라우트 핸들러에서 year 값을 받아온 경우, 해당 year 값에 따라 검색어를 필터링
```jsx
// 게시판 글 검색
router.post("/search", async (req, res) => {
  const { search, year } = req.body;

  // 기본 SQL 쿼리
  let sql = `
    SELECT * FROM board
    WHERE (title LIKE ? OR customer LIKE ?)
  `;

  const params = [`%${search}%`, `%${search}%`];
  
  // year 값이 있는 경우 조건 추가
  if (year) {
    sql += ` AND year = ?`;
    params.push(year);
  }


  try {
    const data = await query(sql, params);
    res.send(data);
  } catch (error) {
    console.error("검색 중 오류 발생:", error);
    res.status(500).send({ msg: "검색 중 오류가 발생했습니다.", error });
  }
});
```

### search 값이 없고 year 값만 있는 경우

```jsx 
router.post("/search", async (req, res) => {
  const { search, year } = req.body;

  // 기본 SQL 쿼리
  let sql = `SELECT * FROM board WHERE 1=1`; // 기본 조건 (항상 참)

  const params = [];

  // search 값이 있는 경우 조건 추가
  if (search) {
    sql += ` AND (title LIKE ? OR customer LIKE ?)`;
    params.push(`%${search}%`, `%${search}%`);
  }

  // year 값이 있는 경우 조건 추가
  if (year) {
    sql += ` AND year = ?`;
    params.push(year);
  }

  try {
    const data = await query(sql, params);
    res.send(data);
  } catch (error) {
    console.error("검색 중 오류 발생:", error);
    res.status(500).send({ msg: "검색 중 오류가 발생했습니다.", error });
  }
});
```
